# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing security@example.com. We take all security issues seriously and will work quickly to address any vulnerabilities that are identified.

When reporting a vulnerability, please include the following information:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact of the vulnerability
- Your name and contact information (optional)

We appreciate your efforts to responsibly disclose any security problems you find, and we will make every effort to acknowledge your contributions.

The following versions of the project are currently being supported with security updates:
